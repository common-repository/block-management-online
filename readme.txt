This plugin integrates the Block Management Online property management and leaseholder web portal within your Wordpress website.

== Description ==

The Block Management Online plugin for Wordpress integrates your leaseholder web portal seamlessly with your
Wordpress website, allowing owners and residents secure access to documents and notices relating to your properties.

For support please visit the [Block Management Online](http://www.blockmanagementonline.com "Block Management Online") website

Although this plugin is free, the use of Block Management Online requires a separate license. Please see the website
for more details about features and pricing.

Copyright (c) 2013 Esscotti Ltd

== Installation ==

1. Copy the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a page and put a [bmo-portal] shortcode on it; (this is the portal page).
4. Visit the Block Management Online admin panel and set the relevant options.
